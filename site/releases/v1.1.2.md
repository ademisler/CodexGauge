# CodexControl 1.1.2

## Highlights

- Fixed the packaged Windows switch-account regression that prevented Codex Desktop from restarting into the newly selected session
- Added a regression test that protects the hidden PowerShell launcher behavior

## Windows Fix

- Removed `DETACHED_PROCESS` from the hidden PowerShell restart helper launch flags
- Kept the direct hidden PowerShell path so packaged restarts still avoid a visible console flash
- Added coverage that asserts the launcher keeps `CREATE_NO_WINDOW` and `CREATE_NEW_PROCESS_GROUP` while excluding `DETACHED_PROCESS`
